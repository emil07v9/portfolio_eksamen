Tema 4 - Grundlæggende Animation

I tema 4 blev vi introduceret til aktivitetsdiagrammer, statemachine diagrammer, skitseringsteknikker og idegenereringsteknikker. Derudover fik vi grundlæggende viden om spildesign som figur-elementer, UI-elementer, start og slutskærm samt baggrundsdesign. Endvidere udviklede vi vores færdigheder inden for CSS-animation og tilmed blev vi introduceret til javascript. Under dette tema brugte vi Adobe programmerne Audition og Illustrator udover de tidligere nævnte programmer. Alt dette implementerede vi i vores afsluttende spil-dokumentationswebsite med tilhørende link til eget fungerende spil.

Spil og dokumentationswebsite

Opgavebeskrivelse

I denne opgave skulle vi udtænke, designe og programmere vores eget spil. Efterfølgende skulle vi opsætte et website, som præsenterede vores spil samt tilhørende assetliste og dokumentation.

I idé og design udviklingsfasen producerede vi paper/lofi prototypes og skitser samt aktivitetsdiagrammer for spillet. Derefter rentegnede vi figurer og skærme i Adobe Illustrator og definerede farver for spillet.

Efter designprocessen gik vi i gang med kodningen af spillet ved brug af HTML, CSS og Javascript. Her lærte vi om CSS positionering og Transform samt CSS animationer, og hvordan man implementerer dem gennem javascript. I javascript arbejdede vi blandt andet også med animationend, animationiteration reflow, konkatenering som random position mm. Under kodningen udarbejdede vi også et state machine diagram over spillet.

Til sidst i projektet arbejdede vi kort med lyd og optageteknik i form af lydeffekter til spillet, som vi selv skulle optage og redigere i Adobe Audition. Efter implementering af lyde kunne vi opsætte websitet for spillet, hvorpå vi samlede al dokumentation fra projektet.
